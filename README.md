# Mpumalanga Tourism Website

## Overview

This project is a responsive tourism website created using **HTML5** and **CSS3**.
The website promotes tourism in Mpumalanga Province, South Africa, by showcasing scenic attractions, cultural information, climate conditions, and visitor services.

The project was developed using semantic HTML structure and responsive CSS layout techniques.

---

# Features

* Responsive website layout
* Semantic HTML5 structure
* Navigation menu
* Tourist attraction gallery
* Population and climate information
* Registration form
* Contact page with Google Maps integration
* Styled tables and forms
* Mobile-friendly design

---

# Pages Included

## Home Page

Displays information and images of famous attractions including:

* God's Window
* Lisbon Falls
* Echo Caves
* Blyde Canyon
* God's Finger

## Population & Climate Page

Contains:

* Cultural population statistics
* Seasonal weather information
* Climate tables

## Registration Page

Includes:

* Visitor registration form
* Food preference selection
* Additional visitor information form

## Contact Page

Contains:

* Contact information
* Contact form
* Embedded Google Map location
